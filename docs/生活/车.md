# 车

## 二手车

- 螺丝喷漆：机舱内固定翼子板螺丝和固定发动机盖螺丝，拆卸过程中会磨去原本喷漆。

- 底盘：检查锈蚀情况（手机超广角车底录像）

- 动力系统：

- 维保记录：按时更换机油、按里程进行大保养（正时皮带、变速箱油、冷却液）。

- 油水：拿纸巾擦油尺看浑浊度。发动机机油、转向机助力油、前差速器油、变速箱油、刹车油（含水量）、发动机冷却液水位

- 检查备胎仓有无钣金痕迹，有无新的胶痕，如果有的话，说明这台车曾经被严重追尾，大概率已经伤到后防撞钢梁了。

- 检查全车玻璃的生产日期，如果日期全部相近，那说明未更换过玻璃，基本可以确认没出过大事故，但不排除少部分车商为了隐瞒事故故意换拆车件，因此生产日期相近不等于没换过玻璃。

- 水泡车和火烧车的痕迹通常更隐蔽，需要拆卸内饰进行辨别，建议与车商沟通并聘请专业人员进行检查。

### 试车

- 确认车子能否一把着，如果不能的话需要考虑蓄电池和起动机是否有故障。

- 将车子从停车位开出来后检查地面上是否有油污，以确认发动机是否有严重渗漏。

- 待水温上升到正常水位后（水温表中间），靠边停车，挂空挡，然后地板油，看发动机能否顺畅的从怠速的800转一路升到断油（一般是4500-5500转断油），重复三次，然后把车挪开，检查地面是否有油污，检验发动机在极端工况下是否有渗漏。

- 如果是自动挡的车子，再次靠边停车，然后在RND三个档位之间来回切换，至少8次，在此期间检查变速箱警告灯是否点亮，如果不亮，再仔细确认换挡时车子有无闯动现象（即前后晃动）

- 起步，控制车速在20-60公里之间波动，检查变速箱升降档时是否平顺，有无顿挫现象
