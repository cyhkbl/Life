# 鳕鱼

## 鳕鱼属（Gadus）

太平洋鳕鱼（Gadus macrocephalus）（真鳕、大头腥）：EPA+DHA ≈ 0.25–0.35 g，汞 0.05–0.08 mg（低汞），蛋白质 18 g，脂肪 0.7 g。小刺极少。鱼白像脑花的形状，雪白细嫩

大西洋鳕鱼（Gadus morhua）：EPA+DHA ≈ 0.3–0.4 g，汞 0.05–0.08 mg（低汞），蛋白质 18 g，脂肪 0.7 g。小刺极少。和太平洋大头腥长得很像，只是斑点更细小一些

## 狭鳕属（Theragra）

黄线狭鳕（Theragra chalcogramma）（明太鱼）：EPA+DHA ≈ 0.2–0.3 g，汞＜0.05 mg，脂肪 0.5 g，蛋白质约16–18g/100g。小刺极少。鱼肉洁白细嫩，适合做炸鱼排。

## 非鳕鱼属

裸盖鱼（Anoplopoma fimbria）（银鳕鱼/黑鳕鱼）：EPA+DHA ≈ 1.3–1.8 g，汞 0.2 mg 左右。无小刺。虽然不是真的鳕鱼，但是肉质更好，价格更贵

小鳞犬牙南极鱼（dissostichus eleginoides）、莫氏犬牙南极鱼（dissostichus mawsoni）（法国银鳕鱼）：EPA+DHA ≈ 1.5–2.0 g，汞 0.3–0.4 mg（中高汞），蛋白质约18–20g/100g。无小刺。肉质更好，价格也比鳕鱼和银鳕鱼更贵

大西洋庸鲽（Hippoglossus hippoglossus）、太平洋产狭鳞庸鲽（Hippoglossus stenolepis（扁鳕鱼/比目鱼）：EPA+DHA ≈ 0.4–0.6 g，汞 0.1–0.2 mg。无小刺。体大肉厚，很扁的鱼排。

棘鳞蛇鲭（Lepidocybium flavobrunneum）异鳞蛇鲭（Ruvettus pretiosus）（油鱼）：含 20–25 % 蜡酯，不能被人体消化，易腹泻；EPA+DHA 仅 0.1–0.2 g。

细鳞壮鳕（Hymenocephalus striatissimus）：几乎没有可测 Omega-3（＜0.1 g）。鳞片疏大，所以去鳞的鱼皮的鱼鳞印记也比银鳕鱼（裸盖鱼和犬牙鱼）要稀疏，而且做熟了之后因为含水量大肉质松散，所以吃起来毫无口感，又水又软又散，有富集重金属的风险
