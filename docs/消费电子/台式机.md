# 台式机(PC)

## 主板

供电会骗人。精粤B760M的供电保护，mos温度飙升到160度还不降频。所以对于小品牌标出的性能指标，最好是自动降低半档看待。当然了，即使这样它们也比一线品牌的最低端产品线有性价比。

## 处理器

### 接口类型

**针脚阵列式（如PGA）：** CPU底部带有针脚，主板上是带孔的插座。

常见平台：AMD台式机CPU的标准（如AM4、AM5），也是英特尔移动平台（笔记本）CPU的常见形式。

**触点阵列式（如LGA）：** CPU底部是平整的金属触点，针脚在主板的插座上。

常见平台：英特尔台式机CPU的标准。

LGA 1700接口:英特尔于2021年底推出的接口，用于第12代（Alder Lake）、第13代（Raptor Lake）和第14代（Raptor Lake Refresh）酷睿台式机处理器。即将被下一代接口（LGA 1851）取代，不建议选购。

**BGA：** 焊死在主板上，不可更换。

常见平台：笔记本、一体机、迷你PC等。

### 功耗

intel的cpu早就抛弃了tdp（热设计功耗）这个东西。12代酷睿之后的产品在intel官网上是不标注tdp的，取而代之的是pbp（Processor Base Power） 和mtp（Maximum Turbo Power）。购买intel的cpu的消费者，在选择主板（供电能力）和散热（解热能力）的时候，**只需要以mtp为参考标准。**

## 内存

关注：容量＞频率。

D5双通道是单条内存把64bit拆分成2x32bit，并不是真正的双通道。诚然，单条D5与双条对比的性能损失，并不像D4那样拦腰斩。但是，去正经一点的厂家官网看的话就会发现D5时代基本不做单条，比如科赋官网D5产品规格里是没有单条的（D4则有）。现在市面上的单条大部分都是商家自己拆的套条。

一般说的一线闪存品牌指的就是该品牌能够自产闪存颗粒，目前全球一共就五大闪存颗粒制造商，他们对应的自有品牌如下：

1. 东芝颗粒：西部数据/闪迪、东芝铠侠/建兴/浦科特

2. 海力士颗粒：海力士/科赋、Solidigm(原intel闪存)

3. 镁光颗粒：镁光/英睿达

4. 三星颗粒：三星

5. 长江颗粒：致钛
