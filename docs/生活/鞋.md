# 鞋

## 中底

**材料与能量回馈：**

**回弹率：** 专业长跑鞋的回弹率通常在**70%以上**，顶级竞速鞋可达**80%-90%**。一般**超临界发泡**技术中底的重量轻、回弹极高。EVA回弹率约50-60%。

**发泡技术：**

- 传统模压发泡：主要用于**EVA** 及其改性材料。

  - 回弹率：通常为 **50%-65%**

  - 密度：较高，通常为 **0.18 - 0.25 g/cm³**

  - 耐久性：一般，材料在长期压缩后容易出现“压痕”，即缓震性能衰减明显。


- 物理发泡 - 注塑发泡：即“超临界发泡”，将固态的**热塑性弹性体**（如TPU、TPEE、PEBA）颗粒与**超临界流体**（通常是氮气N2或二氧化碳CO2）在专用设备中混合。泡孔极其**细小、均匀、致密**。泡孔壁非常薄，形成理想的闭孔结构。

  *   回弹率：极高，达到 **70%-90%**。

  *   密度：约 **0.10 - 0.15 g/cm³**。

  *   耐久性：优异，因为均匀的泡孔结构能更均匀地分散应力，抗衰减能力远强于传统EVA。


  *   常见材料：
      *   **基于PEBA**：Nike ZoomX, Adidas Lightstrike Pro, PUMA Nitro Elite, Saucony PWRRUN PB。
      *   **基于TPU**：Adidas Boost（是其早期代表性技术）， BASF的Infinergy® 颗粒。
      *   **基于TPEE**：李宁“弜”科技。


- 超临界流体一次发泡成型：与注塑发泡的区别在于，它不使用预浸的颗粒，而是直接从原料开始。将**PEBA或TPU等原料**在超临界流体中直接塑化成熔融状态，然后一次性注入模具，在超临界状态下完成发泡和成型。泡孔比注塑发泡**更均匀、更完整、瑕疵更少**。因为它没有颗粒之间的熔接痕，是一个完全均一的整体。

  *   回弹率：能达到理论极限，是目前所有工艺中**最高**的水平。

  *   密度：可以做到**最轻**。

  *   常见材料：
      *   **基于PEBA**：ASICS FF BLAST TURBO+（用于METASPEED系列）， DESCO的“八爪鱼”中底（用于部分国产品牌顶级款）。

​	在预算允许的情况下，优先选择采用 **PEBA基 + 物理发泡** 的鞋款。这是因为TPU物理发泡的重量通常大于PEBA产品。传统EVA则	是在成本和基础保护上的选择。

**密度（硬度）：** **更低密度通常意味着更轻、更软弹**，但也可能意味着耐久性稍差。

**堆叠高度与落差**

*   后跟高度：通常在 20mm - 40mm 之间。更高的堆叠意味着更多缓冲材料，但也可能带来更高的不稳定性。
*   跟趾落差：**后跟高度 - 前掌高度**。这是至关重要的参数。初次尝试可从6-8mm开始。
    *   **8mm-10mm**：传统落差，对小腿和跟腱压力较小，适合后跟或全掌落地的跑者。
    *   **4mm-8mm**：现代流行落差，鼓励中前掌落地，对跟腱要求更高。
    *   **0mm-4mm**：极简或专项训练鞋，对小腿力量要求极高。

**结构技术：**

- 碳纤维板**：**核心作用是提供“推进感”和“抗弯折刚度”**。它不是一个弹簧，而是一个杠杆**。可以关注板的刚度、形状（全掌铲型、匙型）、放置位置。更刚、更翘的板推进感更强，但对跑者力量和技巧要求也更高。

*   尼龙板：比碳板柔性更好，主要提供**稳定性**和**滚动过渡**，推进感弱于碳板。
*   无板：最自然灵活的脚感，依赖纯中底材料回弹。适合恢复跑、轻松跑和对灵活性要求高的跑者。

## 大底

**橡胶覆盖率：** 在关键磨损区域（后跟外侧、前掌）有高覆盖率橡胶贴片，而非全掌覆盖，是为了平衡重量和耐磨。

**橡胶类型：**

*   碳素橡胶：非常耐磨，但较重。常用于后跟等重点区域。
*   发泡橡胶：更轻、更软，提供更好的湿地止滑，但耐磨性稍差。

**花纹：** 深且多向的凸耳花纹提供更好的抓地力，尤其是在湿滑路面。平整的平面更适合公路和跑道。

**测试**

*   **最长的脚趾**应与鞋头留有约**1-1.5厘米**的空隙（约一拇指宽）。
*   按压鞋面最宽处，应感觉**脚部两侧有轻微接触但无硬性挤压**。
*   尝试**抬脚跟**，后跟不应有明显的上下滑动。

## 足弓类型与内旋

**正常足/正常内旋：** 绝大多数“中性”跑鞋都适合。

**低足弓/过度内旋：** 关注 **“支撑”** 参数。查看鞋中底是否有**高密度材料**（如ASICS的Dynamic Duomax）或**几何结构设计**（如Brooks的Guide Rails）来提供内侧支撑。

**高足弓/内旋不足：** 需要**最大缓冲**和**灵活性**的中性鞋。

检验方法：

### 看旧鞋

把你常穿的鞋放在桌面上，看鞋底磨损：

- **鞋底内侧磨损 > 外侧** → 你可能 **足内翻（过度内旋）**
- **鞋底外侧磨损 > 内侧** → 你可能 **足外翻（过度外旋）**

### 湿脚印法

准备一张纸 / 地砖表面，脚弄湿，正常站立踩一下。

看足弓与脚印连接程度：

- **脚印连得较宽，几乎全脚掌接地** → 足弓较低 → 倾向 **足内翻**
- **脚印只显示脚外缘，足弓很空** → 足弓较高 → 倾向 **足外翻**

### 照镜子看脚踝

把镜子放在脚后面，双脚自然站立：

- **脚跟像“／＼”，脚踝向内倒** → **内翻（overpronation）**
- **脚跟像“＼／”，脚踝向外倒** → **外翻（supination）**

### 单脚站立感觉重心

单脚站立，保持自然放松：

- **重心偏脚掌内侧、脚踝容易塌** → **内翻**
- **重心偏脚掌外侧、外侧容易紧绷** → **外翻**

### 蹲下观察膝盖轨迹

双脚与肩同宽，慢慢下蹲：

- 膝盖往 **内侧** 倒 → **足内翻**
- 膝盖往 **外侧** 开 → **足外翻**

## 跑姿与落地方式

**后跟落地**：需要**后跟部位有充足缓冲和良好坡差设计**的鞋。

**中前掌落地**：需要**前掌部位缓冲充足、落差较小**的鞋。

## 整鞋

**单只重量：**

*   训练鞋：**250克 - 300克**
*   轻量训练/竞速鞋：**220克 - 250克**
*   顶级竞速鞋：**< 220克**

## 板材

### 碳板

- **反弹强、更省力**：碳板像弹簧杠杆，提高推进力。
- **中长距离提速明显**：配速 5 分以内的人感受最明显。
- **硬、不够稳**：速度慢的时候反而别扭，容易顶脚和累小腿。
- **容易受伤**：小腿、跟腱负担变大，不适合新手或跑量少的人。
- **寿命较短**：碳板鞋大多寿命在 300–500 km。
- **5–7 配速不太吃到红利**：提速效果几乎感觉不到。

适合：配速 5 分以内或跑 10km/半马

### 尼龙板

- **比碳板更软、更好适应**
- **有轻微推进感，跑起来有点弹**
- **稳定性更好**
- **适合日常训练、5–10km、普通人慢跑**
- **推进力有限**，想提速没碳板明显
- **鞋更像训练鞋，不适合全马比赛冲刺**

适合：想体验一点弹性

### 无板

- **最软、最稳定、对新手最友好、舒适性最好**
- **最不容易受伤**
- **适合 5–7 分配速、5 公里以内的轻松跑**
- **推进力比碳板或尼龙板弱**

适合：放松慢跑、健身、减脂、跑龄不长、偶尔跑 3–5 km
