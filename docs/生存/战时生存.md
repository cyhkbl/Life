# 战时生存

## 战争爆发的预警信号

当观察到以下全部或大部分事件发生时，可简单认定战争准备已经开始：

1.  **舆论动员**：大规模宣传对方国家/民族对我方的历史压迫、经济脱钩等负面信息。
2.  **外交降级**：互相召回大使、撤侨。
3.  **社会动员**：国家号召大规模献血、大规模检查粮仓储备、召回预备役人员。
4.  **经济指标**：钢材、石油等战略物资价格大幅上涨。

## 战前物资准备

### 医疗药物

*   **长期病患**：足量的处方药。
*   **通用药物**：头孢、阿莫西林（抗生素）、碘酒、酒精、止血绷带、烧伤药物、消炎药。

### 生活

*   **净水**：净水片、净水药物、物理式净水器。
*   **食物**：罐头、压缩饼干等易于常温保存的食品。
*   **维生素**：补充营养缺失。
*   **大量黑色加厚垃圾袋**：用于套在桶上解决排泄问题（冲水马桶大概率失效）。
*   **生石灰/猫砂/锯末**：用于掩盖排泄物气味并干燥杀菌。
*   **漂白水（次氯酸钠）**：这是最廉价且强大的杀菌剂，不仅用于环境消毒，极度稀释后也可用于应急净水。
*   **女性生理用品**：在战时是极高价值的硬通货。
*   **湿巾**：在缺水无法洗澡时，维持最低限度的体表清洁。

### 能源与通讯

*   **电力**：手摇发电机或太阳能发电机、大量蓄电池、干电池。
*   **通讯**：短波收音机。
*   **隐蔽**：遮光窗帘或红外阻断毯。
*   **EMP防护**：收音机、手电筒、盖革计数器如果被EMP烧毁就没用了。建议准备一个法拉第笼（哪怕是金属饼干盒内衬纸板，把电子设备放进去）。

### 货币

*   **小面额贵金属**：金豆、银币（金条太大找不开，容易引来杀身之祸）。
*   **现金**：美元、欧元等强势货币的小面额纸币。
*   **高价值消耗品**：香烟（最重要的货币之一）、高度酒、糖、盐、打火机。

### 身份证明

*   **纸质备份**：身份证、护照、房产证、学位证、结婚证、家庭照片。复印多份，过塑防水。
*   **加密U盘**：扫描所有重要文件存入加密U盘，体积小易携带。

### 工业国家战争

若担心核战争/化学武器战争/生物武器战争爆发，除常规战争物资外，需额外准备：

*   **大量瓶装水**：核爆7天内严禁饮用地表水。
*   **盖革计数器**：用于评估辐射风险。
*   **全身防护**：全身包裹的雨衣/一次性雨衣或防化服（防止最初72小时接触空气中辐射尘埃）、墨镜。
*   **防辐射药物**：**专业防辐射碘片（防辐射计量）**（碘盐无效）。
*   **呼吸防护**：防毒面具(3M/Honeywell)或N95以上级别口罩。
*   **密封材料**：大量胶带，用于包裹物体表面防沾染。
*   **密封食品**：防止放射性尘埃污染食物。

*   **应急铝膜毯 (Space Blanket) x2**：反射90%体热。
*   **天幕/雨衣 (Poncho)**：军版雨衣最佳，展开可搭建临时帐篷，隔绝风雨。
*   **打火机 x2**。
*   **打火棒**：在打火机失效、浸水后的终极手段。
*   **凡士林棉球**：作为助燃剂，极其高效。
*   **压缩饼干/能量棒**：无需烹饪，即拆即食。
*   **罐头 (1-2个)**：肉类罐头，提供蛋白质和油脂，提升士气（但因重量大，不宜多带）。

*   **CAT 旋压式止血带 (1-2根)**：这是最重要的救命器材，必须放在背包最容易拿到的地方（或随身携带）。
*   **以色列绷带 (Israeli Bandage)**：多功能压力敷料，用于压迫止血。
*   **胸贴 (Chest Seal)**：用于处理胸部穿透伤（气胸）。
*   **抗生素**：阿莫西林/头孢（需不过敏），应对感染。
*   **止泻药**：蒙脱石散/黄连素。痢疾在战时是致命的。
*   **止痛退烧**：布洛芬/扑热息痛。
*   **碘伏棉签**。
*   **N95/KN95口罩 (3-5个)**：防尘、防病菌。
*   **强力垃圾袋 (5个)**：用于排泄处理、制作雨衣、防水收纳。
*   **免洗洗手液/酒精片**。
*   **头灯 (Headlamp)**：解放双手。务必具备红光模式（夜间阅读地图、低可视度，防暴露）。
*   **备用电池**：锂电池为佳，耐低温。
*   **小型太阳能板/充电宝 (10000mAh)**。
*   **指北针**：物理指针，不依赖电力。
*   **本地纸质地图**：提前打印，标记好撤离路线、水源、汇合点。
*   **记号笔**：用于沿途留记号。
*   **多功能钳 (Multi-tool)**：含刀、钳子、螺丝刀、开罐器。比单纯的匕首更实用。
*   **伞绳 (Paracord) 10-20米**：承重550磅，用于搭建庇护所、捆扎装备、急救固定。
*   **大力胶带 (Duct Tape)**：缠几圈在水壶上节省体积，万能修补工具。
*   **收音机**：全波段收音机（带短波SW），用于接收外界资讯。
*   **口哨**：用于受困时呼救，比喊叫节省体力且穿透力强。
*   **手套**：战术手套或劳保手套。城市废墟中充满了碎玻璃和生锈金属，手受伤意味着丧失劳动和战斗能力。
*   **重要文件U盘与复印件**（防水密封）。
*   **现金/硬通货**。

#### 装包原则

1.  **底部 (Bottom)**：轻且大的物品（如睡袋、衣物）。这可以作为缓冲。
2.  **背部贴合区 (Center/Back)**：**最重**的物品（如水、食物、电台）。遵循重心贴背原则，减少力矩，防止后仰。
3.  **上部与外侧 (Top/Front)**：中等重量、形状不规则物品。
4.  **顶包/腰带/外挂 (Access)**：**高频急用物品**（急救包、雨衣、地图、零食、刀具）。必须做到不卸包即可盲取。

## 战争期间生存

### 位置选择与转移

*   **高危区域**：重要铁路、公路、航空枢纽附近。建议立刻转移，这些区域遭受第一轮攻击概率为100%。
*   **被围困/拉锯战区**：南斯拉夫经验表明，离开被围困城市是首要目标。
*   **转移原则**：关注战线变化（通过短波收音机），随着战线逼近，基础设施会逐渐失能。当所在城市电力系统出现问题时立刻离开。

### 自我保护与武装

*   **可行性方案**：加入一方军队。
*   **热武器禁忌**：**不要携带热武器**。平民身份是最大的护身符，持有武器会被交战双方视为敌人，被清除的概率极大。
*   **防身工具**：扳手、铲子等工具类武器。首选**长柄消防斧**等劈砍类工具（刀具对技巧要求过高）。
*   **团队生存**：最有效的保护是足够大的群体。尽量借助家人、邻居、小区互助组织生存，结队转移。
*   **放弃个人抵抗**：在围城战中，面对现代精准武器，个人抵抗毫无意义，除非加入一方正规军。
*   **被动防御**：防刺服、防弹插板、防弹衣、头盔。

### 信息化

*   **灯火管制与热成像**：现代无人机配备热成像。夜晚生火、使用未遮蔽的手电筒极其危险。
*   **信息静默**：避免在交战区使用手机通话或发推特，信号基站定位可能招致炮火覆盖。
*   **离线数据**：提前下载离线地图（如Maps.me, OsmAnd），下载维基百科离线版（Kiwix），因为互联网随时会断。

### 核战争

*   **反应时间**：约15分钟（飞行时间）。
*   **杀伤形式**：冲击波、热辐射、原始粒子辐射、核电磁脉冲（均为瞬间爆发）、残留放射性（持续性）。
*   **目标预测**：建议使用 *NUKEMAP by Alex Wellerstein* 网站模拟。
    *   **炸点选择**：机场、政府、重要设施、人口密集区。
    *   **数量模拟**：大型城市设5~6个原爆点，中型城市设1~2个。
    *   **参照武器**：美军W88型核弹头。
*   **寻找掩体**：最近的地铁站、地下车库。抵御核辐射的深度最少需要**地下10米**左右。
*   **7/10原则**：核爆后，每过7倍时间，辐射强度下降10倍。这意味着前24-48小时是生死的关键，绝不能离开掩体。
