# 游戏掌机

**掌机系统：**

| 名称           | 基于            | 不可变系统 | 支援 Nvidia 显卡 | 桌面         | 取向               | 特色                                            |
| ------------ | ------------- | ----- | ------------ | ---------- | ---------------- | --------------------------------------------- |
| SteamOS      | Arch Linux    | 是     | 否            | KDE Plasma | 仅供 Steam Deck 使用 | Valve 亲自维护的作业系统。                              |
| HoloISO      | SteamOS       | 是     | 否            | KDE Plasma | x86 电脑与掌机        | 修改 SteamOS 还原映像档而来的系统，试图让一般电脑也能安装 SteamOS。    |
| ChimeraOS    | Arch Linux    | 是     | 否            | GNOME      | x86 电脑与掌机        | 提供简单界面，远端管理 Steam 游戏；除了 Steam 还内建多游戏模拟器。      |
| Garuda Linux | Arch Linux    | 否     | 是            | KDE Plasma | x86 电脑           | 界面很电竞风。                                       |
| Bazzite      | Fedora Atomic | 是     | 是            | KDE Plasma | x86 电脑与掌机        | 提供非常多样搭配的 ISO，可依需求安装适合的版本。                    |
| Nobara Linux | Fedora        | 否     | 是            | GNOME      | x86 电脑与掌机        | Proton GE 的开发者维护的发行版，采用 Linux-Xanmod 内核。      |
| Pop!_OS      | Ubuntu        | 否     | 是            | GNOME      | x86 电脑           | System76 公司专为自家电脑维护的发行版，提供专门支援 Nvidia 显卡的映像档。 |
| Drauger OS   | Ubuntu        | 否     | 是            | XFCE       | x86 电脑与掌机        | 轻量化的发行版；除了 Steam 还内建多游戏模拟器。                   |
