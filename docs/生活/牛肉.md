# 牛肉

## 店铺

- **跨境保税仓直营**：搜索词"保税仓牛肉""跨境牛腱"，查看店铺营业执照+保税仓发货地
- **肉类源头工厂店**：搜索词"XX厂号牛肉"+厂名（如JBS），要求展示厂号授权书+检疫证明
- **批发商分装店**：搜索词"进口牛肉 整条分切"，选评分＞4.8+带「坏果包赔」标
- **品牌旗舰店**：搜索词"大希地""天谱乐食"官方旗舰店，蹲百亿补贴+限量秒杀

## 真伪

- **厂号**+**产地国**+**饲养方式**（例：*"澳203厂谷饲安格斯板腱"*）  
- **检疫标**：放大查看蓝色CIQ标志（入境检验检疫）  
- **厂号贴**：实物包装需有喷码（如*"Plant 969"*）  
- **生产日期**：进口牛肉冷冻保质期约18个月，超1年慎买  
- **脂肪状态**：谷饲牛肉应有均匀雪花纹（商家实拍图需清晰）  
- 单价＜¥20/斤的"进口牛排"多为重组肉（看配料表含**卡拉胶/大豆蛋白**）  
- 拒绝发货地为**河南商丘、山东临沂**的"进口牛肉"（假冒重灾区）

## 部位

- **巴西牛腱**：¥25-32元/斤 
- **乌拉圭牛腩**：¥22-28元/斤       
- **澳洲板腱**：¥40-50元/斤       
- **阿根廷牛尾**：¥35-42元/斤       

## 验货

- **验温度**：泡沫箱内冰袋未全融化（融化率＞80%可退货）  
- **验包装**：扫码CIQ溯源码核对厂号（如969厂对应阿根廷Frigorifico Gorina）  
- **验肉质**：解冻后按压有弹性，无酸腐味（草饲牛肉应呈深红色）
