# 器材

3M Avagard (爱护佳)：粉红色瓶装。接触宠物前后、手部不洁时、进入污染区前。

11号刀片（尖头）：上海金环（刀片）+ 新华（刀柄）。锋利度远超美工刀，适合拆快递、精细手工。

**3M** 胶带：

| 类型 | 3M 型号/俗称 | 优点 | 缺点 | 适用场景 |
| :--- | :--- | :--- | :--- | :--- |
| **纸胶带** | Micropore (透气) | 透气性好，不易致敏，可书写标记 | 粘性较弱，不防水，易脱落 | 敏感皮肤，固定轻型敷料，做标签 |
| **透明胶带** | Transpore (塑料) | 粘性强，固定牢靠，防水 | 撕除时疼痛感强（易带下皮屑毛发），透气性差，部分人过敏 | 固定导管，需要强力固定的部位，非敏感皮肤 |
