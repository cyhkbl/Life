# 汽车事故

1. 出了事故立刻打110和保险公司。对方伤重直接打120，你留在现场等交警。

2. 不要垫付,如果交警要扣车,你就让他扣,所有他需要的资料都给他，15个工作日直接去交警大队要验车报告。不给你，就立刻到交警同一栋办公楼找到行政科提出行政复议，然后拿着验车报告去提车。

3. 停车场不要给钱（拿着验车报告直接就可以拿车），如果对方不给车，直接打110，说有人非法扣车。验车和停车是不要钱的.国家有财拨专门用于这一部分，而且停车场都是外包的。要是担心自己的新车扣了以后被损，就提前拍照，然后用车衣去盖起来。

4. 拿到车后千万不要紧张。现在要做的事情就是不要去医院找打，也不要主动打电话调解。你等他们联系你,或者交警通知你处理事故。任何关于医院的费用，就回答：我没钱，请和我的保险公司联系。如果告我的话，请连我的保险公司一起告。保险公司会请非常专业的律师帮你打官司。

5. 如何拿回自己的行驶证：这个时候对方要么接受调解，愿意接受交警认责比例，把发票给你，你去报销，然后签字，你拿回行驶证。如果对方不愿意调解，那就告，你放心大胆的开车，有人查你就说有官司在身，行驶证抵押了。时间一长，要么保险公司答应赔，要么伤者自己接受调解。法院一判下来，你就可以直接拿判决书去找交警，约对方一起去，签字拿回东西，对方不去，你直接找交警，时间长了不给你行驶证你就去行政科闹一下。

6. 修车：保险勘探现场的时候会有一张现场单，你拿去4S，其它他们会搞定。

7. 营养、误工部分，不要私下答应，当地有一份很完整的赔偿标准，对方拿出所有的发票和证明后交警开具调解书，你拿去保险公司都能报。

8. “三不一没”原则：不垫付、不探望、不调解、没有钱。
